\section{Hypotheses}

This study evaluates two hypotheses:

\begin{itemize}
\item \textbf{H1:} Algorithms with better asymptotic complexity maintain significantly lower runtime growth as input size increases.
\item \textbf{H2:} Memory access patterns that exhibit better cache locality produce significantly lower runtimes than cache-unfriendly patterns, even when algorithmic complexity remains constant.
\end{itemize}
